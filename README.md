## BookVerse

BookVerse is a web app for reading books (PDF/EPUB) and publishing stories.

This repo is being implemented using `implementation_plan_phase0_1.md`.

## Getting Started

1) Install deps

```bash
npm install
```

2) Generate Prisma client

```bash
npx prisma generate
```

3) Run dev server

```bash
npm run dev
```

Open `http://localhost:3000`.
